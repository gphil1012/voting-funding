## Setup
```
git clone https://github.com/capalmer1013/voting-funding.git
cd voting-funding
chmod +x runtTests.sh
chmod +x setupDb.sh
sudo pip install -r requirements.txt
./setupDb.sh
```

To run tests and generate test data for the database run. 

`./runTests.sh`
